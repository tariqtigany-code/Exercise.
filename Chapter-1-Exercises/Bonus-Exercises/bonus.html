HTML & CSS Quick Reference

A guide to common web development tags and properties.

HTML Tags Examples

<h1> to <h6>: Defines headings.
<p>: Defines a paragraph.
<a>: Defines a hyperlink.
<img>: Embeds an image.
<ul> & <li>: Unordered lists.
<ol>: Ordered lists.
<div>: A container for sections.
<span>: Inline container.
<br>: Line break.
<header>: Document header.

CSS Properties Guide

  • color: Sets text color.
  • background-color: Sets background color.
  • font-family: Specifies the font for text.
  • font-size: Adjusts the size of text.
  • margin: Clears area outside the border.
  • padding: Clears area around the content.
  • border: Sets a border around an element.
  • text-align: Aligns the text (left, right, center).
